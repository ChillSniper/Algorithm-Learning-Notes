### build()

```cpp
void build(int u, int L, int R) {
    tree[L].L = L, tree[R].R = R;
    if (L == R) return ;
    int mid = (L + R) >> 1;
    build(2 * u, L, mid);
    build(2 * u + 1, mid + 1, R);
    pushup(u);
}
```
